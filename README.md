# `jotinha`

O `jotinha` é um projeto de código aberto inspirado nas tecnologias e boas práticas de engenharia de software utilizadas pelo [JOTA.info](https://www.jota.info), site que fornece informações de qualidade, com análise e contexto, a um público profissional que precisa tomar decisões estratégicas constantemente com base nas movimentações dos três poderes.

A partir de uma operação enxuta e dinâmica, típica de uma startup, o JOTA desde sempre conecta tecnologia e jornalismo.

Através do `jotinha` disponibilizamos a profissionais de tecnologia acesso a parte dos princípios, práticas e ferramentas que fazem a engenharia de software do JOTA ser especial.

**Algumas das tecnologias e boas práticas que pretendemos disponibilizar no `jotinha`**

- Wordpress headless no back-end
- Front-end desacoplado para consumo do Wordpress headless
- Unittests com phpunit e Wordpress
- CI com GitHub Actions, phpunit e Wordpress
- Testes de componente no front
- Documentação enxuta de software
